日志等级
1. log 通用日志，按需进行记录
2. warning 警告日志，比如尝试多次惊醒数据库操作
3. error 严重日志，比如：
    1. 数据库操作失败
    2. 接口请求失败
    3. 系统异常
4. debug 调试日志，比如：
    1. 接口请求参数
    2. 接口请求结果
    3. 接口请求耗时
5. verbose 详细日志，所有的操作与详细信息（非必要不打印）



日志功能分类：
1. 错误日志 // 方便定位问题，
2. 调试日志 // 方便开发
3. 请求日志 // 记录铭感信息
